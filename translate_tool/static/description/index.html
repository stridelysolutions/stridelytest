Overview

With Translate tool module ,you can Translate the Custom modules or Reports in Odoo by following some quick and easy steps. It saves the time to run the translation as it stores the file in your module location and will access it every time translation is requested.

Screenshots

Open settings

  • Click on Translations

After clicking translations menu then show Translate Tool menu.

By clicking on the 'Translate tool' below screen will be poped up

For example, Driving School is the Custom app we are trying to translate from English to Dutch

Click on Translate Tool Pop Box in select language and apps.Then click on Translate Button

After we Translate the App/Module, system will update it in selected language as shown below.

Some Warnings​

  • If your system do not contain the following libraries mentioned in below Screenshot, then it will show a popup box as shown below at the time of installation​.

  • While translating module if language is not selected then also it will show a popup box to select language in which you want to translate​.

  • If your internet is off then it will show an dialog box that for translating you need to connect to the internet​
